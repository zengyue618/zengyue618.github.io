一个案例快速入门Spark MLlib

一：前言

MLlib是Apache Spark的可扩展机器学习库，是一个分布式的机器学习框架，可用于海量数据的机器学习场景。

今天用一个美国 1994 年人口普查的数据集，通过研究预测居民收入是否超过年收入50万美元这个二分类问题，带大家快速入门Spark MLlib。

这个数据集来源于https://archive.ics.uci.edu/ml/machine-learning-databases/adult/

下载后是.data格式的文件，把后缀改成.csv后，发现长这个样子

一共有15列，其中最后一列是目标变量，其它14列是属性变量。

它们的列名以及含义如下图所示

用于分类的模型有很多

从中，我挑取逻辑回归、决策树、随机森林、梯度提升树、线性支持向量机模型进行演示

用ROC曲线下面积作为评价指标（越大证明模型越好）

处理属性变量

把数值型的属性变量和类别型的属性变量挑出来

numeric = ['age', 'fnlwgt', 'education_num', 'capitial_gain',  'capitial_gain', 'capitial_loss']
category = ['workclass', 'education', 'maritial_status', 'occupation', 
            'relationship', 'race', 'sex', 'native_country']

接下来 要准备先将类别变量转化为数值变量，再将数值变量进行onehot编码，
先后用StringIndexer和OneHotEncoder就可以实现

from pyspark.ml.feature import StringIndexer
category_index = [f'{c}_index' for c in category]
indexer = StringIndexer(inputCols = category, outputCols = category_index)

from pyspark.ml.feature import OneHotEncoder
category_onehot = [f'{c}_onehot' for c in category]
onehoter = OneHotEncoder(inputCols = category_index, outputCols = category_onehot)

而后 将数值变量和转换后的类别变量分别打包为两个变量，
numeric_features和category_features_onehot，
使用VectorAssembler即可

from pyspark.ml.feature import VectorAssembler
numeric_features_assembler = VectorAssembler(inputCols = numeric, outputCol = "numeric_features")
category_features_assembler = VectorAssembler(inputCols = category_onehot, outputCol = "category_features_onehot")

紧接着 要把打包后的数值变量形成的列——numeric_features，
进行min-max标准化，也就是把值映射到0-1中间的数，
用MinMaxScaler完成

from pyspark.ml.feature import MinMaxScaler
minmaxer = MinMaxScaler(inputCol = "numeric_features" , outputCol = "numeric_features_minmax")

准备好了上面的东西后，就按顺序对数据集进行处理

df_spark = indexer.fit(df_spark).transform(df_spark)
df_spark = onehoter.fit(df_spark).transform(df_spark)
df_spark = numeric_features_assembler.transform(df_spark)
df_spark = category_features_assembler.transform(df_spark)
df_spark = minmaxer.fit(df_spark).transform(df_spark)

处理标签变量

对属性变量的处理完成后，就要处理标签变量，
我们对标签列，进行数值化处理即可，使用StringIndexer完成

indexer = StringIndexer(inputCol = "income", outputCol = "label")

df_spark = indexer.fit(df_spark).transform(df_spark)

df_spark = df_spark.select(["category_features_onehot", "numeric_features_minmax", "label"])

三：模型训练

接下来我们就用常见的分类模型，对数据对模型进行训练，
并且使用ROC曲线下面积，作为模型的的评价指标

先把数据集整体划分为训练集和测试集，七三开

train_data, test_data =final_df.randomSplit([0.7, 0.3])

1. 逻辑回归

先用LogisticRegression构建逻辑回归模型，
用fit方法对训练集进行训练，
再用evaluate方法，将测试集应用到训练好模型，得到评价结果，
最后查看ROC曲线下面积

from pyspark.ml.classification import LogisticRegression
lr = LogisticRegression(featuresCol="features", labelCol="label")
lrModel = lr.fit(train_data)
lr_pred_results = lrModel.evaluate(test_data)
lr_pred_results.areaUnderROC

0.9045632288306538

2. 决策树

先用DecisionTreeClassifier构建决策树模型，
用fit方法对训练集进行训练，
以测试集作为输入，对训练得到的模型使用transform方法，得到测试结果，
选取其中的prediction，label两列，前者是模型预测，后者是真实值，

from pyspark.ml.classification import DecisionTreeClassifier
dt = DecisionTreeClassifier(featuresCol="features", labelCol="label")
dtModel = dt.fit(train_data)
dt_transform_test = dtModel.transform(test_data).select("prediction", "label")
dt_transform_test.show(1)

+----------+-----+
|prediction|label|
+----------+-----+
|       0.0|  0.0|
+----------+-----+
only showing top 1 row

​

由于其它分类模型目前没有实现evaluate方法，所以想得到指标得绕一绕

先用BinaryClassificationEvaluator构建二分类专用的评价器
以测试结果为输入，对评价器使用evaluate方法
加上{evaluator.metricName: “areaUnderROC”}参数，得到ROC曲线下面积

from pyspark.ml.evaluation import BinaryClassificationEvaluator
dt_evaluator = BinaryClassificationEvaluator(labelCol = "label", rawPredictionCol = "prediction" )
dt_pred_results = dt_evaluator.evaluate(dt_transform_test, {dt_evaluator.metricName: "areaUnderROC"})
dt_pred_results

0.7276855987873679

四：总结

Spark MLlib的使用方式总体上来说和sklearn比较类似，并且拥有分布式的处理能力。但是目前的模型还没有sklearn那么全。另外，比起sklearn，Spark MLlib用于分类模型训练的数据集需要打包成属性和标签两个列，多了一个打包的过程。如果大家感兴趣，可以到官方文档中，进一步学习。
