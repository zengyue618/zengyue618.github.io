学习完蚂蚁老师的pyecharts数据可视化课程后，受益匪浅！
但是如何让pyecharts的图以自己想要的排版方式显示在网页上呢？
就像这样
看完这篇文章你就能学会！

一：将你的生成pyecharts可视化图像的相关代码包装成一个函数。

类似这样的结构

def func() -> 图的对象名字
  构造对象
  return 对象

下面的代码是生成一个WordCloud词云的对象，并且返回这个对象。

def word_cloud() -> WordCloud:
 c = (
  WordCloud()
   .add("", word, word_size_range=[20, 100], shape=SymbolType.DIAMOND)
   .set_global_opts(title_opts=opts.TitleOpts(title="快时尚行业销售排行"))
 )
 return c

二：从 pyecharts.charts 导入 Page

from pyecharts.charts import Page

写完画图的函数之后，加上这几行代码

page = Page(layout=Page.DraggablePageLayout, page_title="商户视角——行业分析")

page.add(
 word_cloud(),
 bar_tool(),
 scat(),
 uVSz(),
)

page.render('商户视角——行业分析.html')
page_title的值你可以随便改
page.add()里面填的就是你自己写的画图函数，按上面的格式添加进去
最后 page.render(‘XXXX.html’) 这行代码之后过后会生成一个名叫XXXX的html文件

打开这个文件，里面就是仪表板的初始效果了

三：根据你的意愿，对图片进行排版

你可以用鼠标点击图片拖拽，可以用滚轮缩放图像比例等等

四：保存你排版好的html文件

点击 它就会自动生成一个json文件 把这个json文件放到你代码的目录下

五：形成自定义布局

Page.save_resize_html("源.html", 
 cfg_file="chart_config.json",
  dest="自定义布局.html")

执行上述代码，三个参数分别是，原先生成的html文件，保存布局产生的json文件以及你想要生成的自定义布局的html文件

最后打开dest参数指定的html文件，你就会发现做好了一张由你布局的仪表板啦！
